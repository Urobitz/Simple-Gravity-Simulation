## Simple-Gravity-Simulation
This program is a 2D gravitional simulation built on C++ using SFML graphics. The project is used to model simple interactions between objets and attractors using a simplified gravitational force system. 

## Controls
**Left click** -> Add attractor with static mass (Can be changed editing ATTRACTOR_MASS).
**Right Click** -> Add a body with a random mass.

## Features

- Newtonian gravitational simulation.
- Numerical Integration using Euler Method.
- Logarithmic mass scaling for visuals.
- Dynamic creation of both bodies and attractors.
